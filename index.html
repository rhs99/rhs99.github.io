Xv6 2024

C

Xv6 is a UNIX like OS for instructional purpose. Implemented unix utilities like sleep, find, xargs, pipe and system calls like trace, sysinfo and pgaccess. Reduced execution time for certain system calls by sharing data in a read-only region between userspace and the kernel. Implemented debugging functionalities like printing page tables, showing function call stack. Added a feature to periodically alert a process as it uses CPU time. Implemented copy on write (COW) fork
